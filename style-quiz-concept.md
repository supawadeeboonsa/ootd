## 7 Style Archetypes

| Style | คำนิยามสั้นๆ | Color Identity |
|---|---|---|
| **Minimal** | ตัดทุกอย่างที่ไม่จำเป็นออก เหลือแค่สิ่งที่ดีที่สุด | ⬜⬛🩶 |

## Style Quiz — 10 ข้อ

ออกแบบให้แต่ละข้อวัด **dimension ที่ต่างกัน** ไม่ใช่แค่ถามซ้ำๆ

---

### ข้อ 1 — Energy / Vibe
**"เวลาเดินเข้าห้อง คุณอยากให้คนรู้สึกยังไงกับคุณ?"**

```
A  ไม่ต้องรู้สึกอะไรพิเศษ — คุณแค่อยู่ตรงนั้น      → Minimal, Old Money
B  อยากให้คนสังเกตเห็นแต่ไม่รู้ว่าทำไม            → Korean Casual, Smart Casual
C  อยากให้คนรู้สึก energy ของคุณก่อนเดินเข้ามา    → Streetwear
D  อยากให้คนสงสัยว่าเสื้อมาจากไหน               → Vintage
```

---

### ข้อ 2 — Color Comfort Zone
**"Palette ที่คุณหยิบมาใส่โดยไม่คิดคือ?"**

```
A  ขาว เทา ดำ น้ำตาล — สีที่ mix กับทุกอย่างได้     → Minimal, Old Money
B  ขาว ครีม ชมพูอ่อน น้ำเงินซีด — pastel territory  → Korean Casual
C  ดำ เทาเข้ม กรมท่า พร้อม pop 1 สี                → Streetwear, Smart Casual
D  สีที่มีประวัติศาสตร์ — mustard terracotta olive  → Vintage
```

---

### ข้อ 3 — Fit Philosophy
**"ทรงเสื้อผ้าที่คุณรู้สึก 'ใช่' มากที่สุดคือ?"**

```
A  Tailored — ทรงชัด ทุกอย่างอยู่ที่มันควรอยู่      → Minimal, Old Money, Smart Casual
B  Oversized ทับ Fitted — เล่น proportion           → Streetwear, Korean Casual
C  Relaxed แต่ไม่หลวม — comfortable first           → Korean Casual, Smart Casual
D  อะไรก็ได้ถ้า texture หรือ pattern มันน่าสนใจ     → Vintage
```

---

### ข้อ 4 — Brand Relationship
**"คุณรู้สึกยังไงกับ logo และ branding บนเสื้อผ้า?"**

```
A  ไม่อยากให้มี ยิ่งเงียบยิ่งดี                     → Minimal, Old Money
B  โอเคถ้าเป็น subtle logo เล็กๆ                   → Smart Casual, Korean Casual
C  Logo และ graphic คือส่วนหนึ่งของ look            → Streetwear
D  ไม่สน brand — สนแค่ว่าของชิ้นนั้นมีเรื่องราวไหม  → Vintage
```

---

### ข้อ 5 — Getting Ready
**"เวลาแต่งตัว process ของคุณเป็นยังไง?"**

```
A  รวดเร็ว — หยิบ 3 ชิ้น ออกไปได้เลย               → Minimal
B  ใช้เวลา layering และทดลอง combination            → Korean Casual, Vintage
C  วางแผนล่วงหน้า รู้ว่าจะใส่อะไรตั้งแต่เมื่อคืน    → Old Money, Smart Casual
D  อยู่กับ vibe ของวัน — spontaneous                → Streetwear
```

---

### ข้อ 6 — Occasion Flexibility
**"ถ้าต้องเลือกชุดชุดเดียวไปงาน 3 อย่างในวันเดียว (กาแฟเช้า / ประชุม / dinner) คุณจะ?"**

```
A  เลือกชุด neutral ที่ดูดีในทุก context            → Minimal, Smart Casual
B  เพิ่ม accessory และ layer ระหว่างวัน             → Korean Casual
C  ยึด style ตัวเองไว้ ปล่อยให้คนอื่น adapt         → Streetwear, Vintage
D  เลือก classic piece ที่ทำให้ดูน่าเชื่อถือทุกที่   → Old Money
```

---

### ข้อ 7 — Material & Texture
**"เวลาเลือกเสื้อผ้า มือคุณสัมผัสผ้าก่อนเสมอ — คุณชอบ?"**

```
A  ผ้าที่น้ำหนักเบา smooth ไหล — ดูแลง่าย           → Minimal, Korean Casual
B  Cotton หนา fleece canvas — durable และ honest    → Streetwear
C  Wool cashmere linen — natural และมีอายุดีขึ้น    → Old Money
D  ผ้าที่มีเรื่องราว — tweed denim เก่า ผ้าทอ       → Vintage
```

---

### ข้อ 8 — Reference World
**"สิ่งที่ inspire การแต่งตัวคุณมากที่สุดคือ?"**

```
A  สถาปัตยกรรม การออกแบบ art direction             → Minimal
B  ดนตรี ศิลปิน culture ที่คุณชอบ                  → Streetwear
C  คนที่แต่งตัวดีในหนังเก่า series period drama    → Old Money, Vintage
D  GRWM / street snap / everyday fashion content   → Korean Casual, Smart Casual
```

---

### ข้อ 9 — Wardrobe Size Philosophy
**"Wardrobe ในฝันของคุณคือ?"**

```
A  30 ชิ้น แต่ทุกชิ้นสมบูรณ์แบบ — capsule wardrobe  → Minimal, Old Money
B  เต็มไปด้วยชิ้นที่หาไม่ได้ที่อื่น — curated chaos  → Vintage, Streetwear
C  Mix ระหว่าง basic และ statement piece            → Korean Casual, Smart Casual
D  ไม่สน size สน vibe — ถ้าชอบก็เอา                → Streetwear, Vintage
```

---

### ข้อ 10 — Self-expression
**"สำหรับคุณ การแต่งตัวคือ?"**

```
A  Armor — ช่วยให้รู้สึก ready และมั่นใจ            → Minimal, Smart Casual
B  Art — expression ที่เปลี่ยนได้ทุกวัน              → Vintage, Korean Casual
C  Identity — บอกว่าฉันเป็นส่วนหนึ่งของ culture นี้  → Streetwear
D  Habit — ทำจนชินแล้วก็ไม่ค่อยคิดเรื่องนี้          → Old Money
```

## ระบบวิเคราะห์คำตอบ

### Scoring Matrix

แต่ละคำตอบให้คะแนน 6 สไตล์ (0–2 คะแนน):

```
ตัวอย่าง ข้อ 1:
  A → Minimal +2, Old Money +1
  B → Korean Casual +2, Smart Casual +1
  C → Streetwear +2
  D → Vintage +2
```

### ประเมินผล

```
คะแนนรวมสูงสุด (max 20 ต่อ style)
→ Primary Style   = อันดับ 1
→ Secondary Style = อันดับ 2 (ถ้า gap < 3 คะแนน = hybrid)

ตัวอย่างผล:
  Minimal 14  |  Korean Casual 11  |  Old Money 8  ...
  → Primary: Minimal | Secondary: Korean Casual
  → Label: "Minimal with Korean flair"
```

### Hybrid Labels (เมื่อ 2 สไตล์ใกล้กัน)

| Combination | Label |
|---|---|
| Minimal + Old Money | **Quiet Luxury** |
| Minimal + Korean Casual | **Clean Seoul** |
| Streetwear + Vintage | **Archive Culture** |
| Old Money + Smart Casual | **Dressed Down Classic** |
| Korean Casual + Vintage | **Thrift & Tender** |
| Smart Casual + Minimal | **Understated Pro** |

## Mood Image ของแต่ละ Style

รูปที่โชว์ใน result ไม่ใช่ outfit แต่เป็น **aesthetic collage** — atmosphere

| Style | Mood Image Elements |
|---|---|
| **Minimal** | โต๊ะไม้สีอ่อน, แก้วน้ำใส, หนังสือ 2 เล่ม, ผ้า linen ขาว |
| **Streetwear** | sneaker ด้านข้าง, เส้นกราฟิก, urban texture, hoodie พับ |
| **Old Money** | สมุดหนัง, นาฬิกา analog, ผ้าสีครีม, รั้วสวน |
| **Korean Casual** | กาแฟ, plant เล็ก, ผ้า pastel, afternoon light |
| **Smart Casual** | laptop เปิด, กระเป๋า leather, เมืองผ่านกระจก |
| **Vintage** | วินิล, กล้องฟิล์ม, fabric pattern เก่า, แสงอุ่น |

## Card ที่บันทึกได้ — ฟีเจอร์เสริม

เพราะ OOTD เน้นการรู้จักตัวเอง ไม่ใช่การซื้อ การ์ดจึงบันทึกได้หลายแบบ:

| การ์ด | เนื้อหา | ใช้ทำอะไร |
|---|---|---|
| **Style Result Card** | ผล quiz + mood image | แชร์ / เก็บ reference |
| **Color Palette Card** | สี 4–5 สีที่เหมาะกับ style | ใช้ guide ตอนซื้อเสื้อผ้า |
| **Fit Guide Card** | ทรงที่เหมาะ ทรงที่ควรหลีกเลี่ยง | ใช้ตอนเลือกชิ้น |
| **Capsule Checklist Card** | รายการ wardrobe essential ของ style นั้น | checklist ใช้จริง |

## Retake Logic

Quiz ควรทำใหม่ได้ และผลควรเปลี่ยนได้:

```
ทำครั้งแรก → เก็บผล + วันที่
ทำอีกครั้งหลัง 3 เดือน → เปรียบเทียบ: "Style คุณเปลี่ยนไปไหม?"
  ├── ยังเหมือนเดิม → "คุณ consistent มาก"
  └── เปลี่ยนไป    → "Style คุณกำลัง evolve →  Minimal → Old Money"
```

## สิ่งที่ Quiz นี้ **ไม่ทำ**

- ❌ ไม่แนะนำให้ซื้ออะไร
- ❌ ไม่บอกว่า style ไหน "ดีกว่า"
- ❌ ไม่มีคำตอบผิดถูก
- ❌ ไม่ judge ถ้าผลออกมาเป็น hybrid แปลกๆ
- ❌ ไม่บังคับให้อยู่ใน box — "คุณอาจเป็น Vintage ที่มีหัวใจ Streetwear ก็ได้"
